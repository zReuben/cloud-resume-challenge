This project began as the Cloud Resume Challenge, where I implemented a visitor counter backed by a RESTful API (API Gateway ‚Üí Lambda ‚Üí DynamoDB) and deployed a static site on CloudFront/S3 with HTTPS. Everything is defined as Infrastructure as Code using AWS CloudFormation / SAM.

I then extended it into a multi-account AWS setup (mgmt, eng, test, prod) with GitHub Actions CI/CD, environment protections (approvals for test/prod), and guardrails for cost & security. It mirrors how cloud-native teams ship: build once, promote safely, and roll back quickly when needed.
